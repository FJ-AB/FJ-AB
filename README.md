## 🚀 Sobre mí

Soy ingeniero de software con más de 5 años de experiencia, especializado en desarrollo backend, bases de datos y aplicaciones de entorno Windows.

Disfruto especialmente construyendo soluciones desde cero, donde puedo definir una base sólida desde el inicio. Aun así, también tengo experiencia en evolución, mantenimiento y mejora de sistemas existentes.

Mi enfoque del desarrollo va más allá del código:  
👉 creo firmemente que **una buena planificación lo es todo**.

Dedico especial atención a:
- 📋 Análisis y documentación de requisitos  
- 🤝 Comunicación clara con el cliente  
- 🧠 Comprensión profunda del problema antes de implementar  

Entiendo que el software evoluciona constantemente, pero una buena base permite adaptarse mejor al cambio, ser más resolutivo y reducir la complejidad en fases posteriores.
